# AutoResearch UI

Frontend application for the AutoResearch SDPA model.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Backend

This frontend connects to the AutoResearch SDPA inference API.

For the backend implementation and training code, see: [https://github.com/TerkaSlan/autoresearch-win-rtx](https://github.com/TerkaSlan/autoresearch-win-rtx)

## Deployment

The frontend is deployed via Docker and Kubernetes. Deployment manifests are managed separately.
